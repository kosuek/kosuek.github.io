このページについて

このページは、kosukeがこれまでに作ったPWA（Progressive Web App）を見やすくまとめたポータルです。各カードから デモ（公開URL） と GitHub へ移動できます。

  • タグでの絞り込み、テキスト検索に対応
  • カードの順序は apps 配列の並び順で制御
  • アイコン画像は icon プロパティ（正方形推奨）
新しいアプリを追加するには、下の // apps 配列にオブジェクトを1行足すだけです。
